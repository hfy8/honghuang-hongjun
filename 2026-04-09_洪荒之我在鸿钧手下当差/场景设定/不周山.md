# ⛰️ 不周山
> **⚠️ 风格统一：** Anime动漫风格 · 中国古典仙侠风 · 禁止混搭

---

## 场景概述

| 项目 | 内容 |
|------|------|
| **名称** | 不周山 |
| **位置** | 洪荒大地中央，天地之间，北接天穹 |
| **类型** | 盘古遗迹/天地支柱（已倒塌）/凶险禁地 |
| **核心视觉** | 黑色巨峰折断如尸骸，血色煞气冲天，山体布满巨大裂缝如骨骼裂纹 |
| **色调** | 黑色/暗红/灰紫为主，煞气呈血红色，天空晦暗如死灰 |
| **氛围** | 苍凉、悲壮、毁灭与神秘并存、压抑窒息、古老原始 |

---

## 空间布局

| 维度 | 描述 |
|------|------|
| **纵向** | 原高不知几万丈，折断后现存山体仍高约三千里，峰顶直入九天 |
| **横向** | 山体方圆约八百里，根基深入地壳，峰巅破碎如尸骨 |
| **层次** | 山脚（碎石与巨骸）→ 山腰（煞气弥漫）→ 山顶（折断处，劈开天光） |
| **断裂处** | 山峰中段有一巨大裂口，宽度约数十里，如被神斧劈开 |

---

## 建筑外观

### 整体山形

| 项目 | 描述 |
|------|------|
| **外形** | 原为连接天地的巨柱，折断后呈半截残柱形态，西侧仍完整，东侧崩裂 |
| **山体颜色** | 通体黑灰色岩石，表面有岁月侵蚀的纹理，近似骨骼的质感 |
| **山体裂纹** | 无数巨大裂缝从折断处向下延伸，裂缝宽度由数丈到数十丈不等 |
| **残骸** | 山脚散布巨石，形状如肋骨/脊椎/碎骨，最大者高约数十丈 |

### 岩石细节

| 项目 | 描述 |
|------|------|
| **岩石颜色** | 暗黑色，部分呈铁锈红，部分呈青灰色，有熔岩色泽 |
| **岩面纹理** | 表面粗糙，有熔岩流过的痕迹，有天然符文若隐若现 |
| **山壁** | 峭壁如削，壁面有天然刻痕如肌肉纹理，故称"盘古之躯" |

| 项目 | 描述 |
|------|------|
| **位置** | 山体裂缝内壁 |

| **意义** | 据说是盘古临死前的最后意志所化 |

### 煞气结晶

| 项目 | 描述 |
|------|------|
| **位置** | 裂缝深处 |
| **形态** | 暗红色结晶体，贴在岩壁上 |
| **效果** | 散发血红色雾气，触之即被煞气入侵 |

## 天空/气象

| 项目 | 描述 |
|------|------|
| **天色** | 常年晦暗，被煞气遮蔽，不见天日，呈深紫色如黄昏末日 |
| **云层** | 不规则旋涡状乌云在山顶聚集，云层呈暗红色内缘 |
| **雷电** | 山顶折断处常有雷电交错，从裂缝中向外劈出 |
| **光线** | 无正常日光，唯一的光源是裂缝中的血红色煞气光芒 |

## 地面/环境

### 山脚

| 项目 | 描述 |
|------|------|

| **煞气浓度** | 山脚煞气稍淡，但仍浓郁，血雾在地面流淌如河 |
| **声音** | 煞气在石缝中通过，发出呼啸声如泣声 |

### 植被

| 项目 | 描述 |
|------|------|
| **状况** | 不周山方圆千里不生草木 |

### 水景

| 项目 | 描述 |
|------|------|
| **血池** | 山腰有一处天然血池，方圆数十丈，池水如血，沸而不腾 |
| **雾气** | 地面有淡淡血雾流淌，如血河在石缝间穿梭 |

## 重要地点详述

### 折断处（山顶）

- 不周山最震撼之处，巨峰从中断开，断面如被神斧劈开般平整
- 从此处可看到天空的裂缝，即"天破之处"
- 女娲补天前，天河之水从此处倾泻而下

### 血池（山腰）

- 祖巫修炼之地，煞气最为浓郁
- 云霄曾在此处与后土相遇

### 盘古遗骸

- 传说山中有盘古遗留的意志，但凡人无法接近
