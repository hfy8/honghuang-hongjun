## 场景概述

| 项目 | 内容 |
|------|------|
| **名称** | 妖族天庭 |
| **位置** | 三十三天之上，天庭宫阙，九天之内 |
| **类型** | 妖族政治中心/帝俊太一共治/统御万妖之地 |
| **核心视觉** | 金碧辉煌，万丈金光，宫阙连绵如云，山河社稷皆在足下 |
| **色调** | 金色为主，白玉为辅，朱红点缀，日光下金芒万丈 |
| **氛围** | 奢华霸气、君临天下、威严赫赫、皇权至上、不可一世 |

## 空间布局

| 维度 | 描述 |
|------|------|
| **纵向** | 天庭依天势而建，高约九重，层层向上，最高层直抵凌霄宝殿 |
| **横向** | 天庭横跨数万里，宫阙连绵，东至汤谷，西至昧谷 |
| **层次** | 南天门（入口）→ 外朝（百官朝会）→ 内朝（帝俊寝宫）→ 凌霄宝殿（最高点） |
| **轴线** | 以凌霄宝殿为中轴线核心，宫阙左右对称分布 |

## 建筑外观

### 屋顶

| 项目 | 描述 |
|------|------|
| **样式** | 重檐庑殿顶与攒尖顶并用，重要殿宇用重檐，标志性建筑用金顶攒尖 |
| **材质** | 赤金瓦，瓦片以纯金锤炼，厚约一指，在日光下璀璨耀目 |
| **脊饰** | 正脊以金铜铸日月星辰，脊中立金乌展翅，高约二丈 |
| **檐角** | 檐角铜铃改为金铃，每角一串，风起时金铃齐鸣如天籁 |

### 墙体

| 项目 | 描述 |
|------|------|
| **材质** | 以白玉为基，外贴金砖，金砖表面铸有金乌/凤凰/腾蛇等妖族圣兽纹样 |
| **颜色** | 日间金芒万丈，墙面如镜反射日光，夜间金砖自身散发柔光 |
| **高度** | 宫墙高约七丈，厚度约一丈五尺 |

### 天门

| 项目 | 描述 |
|------|------|
| **数量** | 四座天门，分别为南天门（正门）、东天门、西天门、北天门 |
| **南天门高度** | 高约三十六丈，宽约二十四丈，以金铸成，门楣高悬"天庭"二金字 |
| **门柱** | 以白玉为柱，柱身盘九条金龙，龙首在柱顶，龙口衔金珠 |
| **门楼** | 五层门楼，每层飞檐翘角，最高层檐角悬金灯，共十二盏 |

### 台阶

| 项目 | 描述 |
|------|------|
| **数量** | 自南天门至凌霄宝殿，共计九层台阶，每层九十九级 |
| **材质** | 以金丝楠木铺面，边缘嵌白玉，高贵庄重 |
| **宽度** | 主台阶宽约九丈，可并行十二人 |
| **两侧** | 台阶两侧立金甲神将，共一百零八位，手持各式兵器 |

## 重要地点详述

### 凌霄宝殿

- 天庭最高点，帝俊朝会百官之地
- 殿内中央有帝俊宝座，以金铸成，镶嵌宝石
- 殿顶有金乌盘旋，光芒万丈

### 东皇宫

- 太一居所，与凌霄宝殿并重
- 殿内以太阳真火为灯，以金乌为饰

### 御花园

- 帝俊休憩之地
- 园中有不死药圃，珍稀花卉无数
