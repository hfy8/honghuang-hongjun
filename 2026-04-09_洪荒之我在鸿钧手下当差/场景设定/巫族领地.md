# ⛰️ 巫族领地
> **⚠️ 风格统一：** Anime动漫风格 · 中国古典仙侠风 · 禁止混搭

---

## 场景概述

| 项目 | 内容 |
|------|------|
| **名称** | 巫族领地 |
| **位置** | 洪荒大地东南区域，崇山峻岭之中 |
| **类型** | 巫族聚居地/十二祖巫统领/盘古后裔根据地 |
| **核心视觉** | 巨石堆砌的粗犷殿堂，巨兽骨骼搭建的图腾柱，血色火焰长燃，古老而野蛮 |
| **色调** | 暗红/黑褐/灰石为主，火光为唯一光源，血色点缀 |
| **氛围** | 粗犷原始、野性血腥、古老神秘、原始力量、崇拜盘古 |

---

## 空间布局

| 维度 | 描述 |
|------|------|
| **纵向** | 依山势而建，由山脚延伸至山腰，层次分明 |
| **横向** | 领地广袤，东西约数千里，南北亦数千里 |
| **层次** | 山脚（血池/祖火）→ 山腰（祖巫宫殿）→ 山顶（祭台） |
| **布局** | 以祖巫殿为核心，各祖巫领地呈扇形分布 |

## 建筑外观

### 祖巫殿

| 项目 | 描述 |
|------|------|
| **整体** | 以巨型石块堆砌，石块大者可达数丈，小者亦逾一人高，堆砌不留缝隙 |
| **高度** | 主殿高约九丈，宽约十二丈，进深约十五丈 |
| **屋顶** | 以整块巨石板为顶，板厚约三尺，板上刻有祖先符文 |
| **门洞** | 以巨兽骨骼搭建门框（门框高约四丈），上挂兽皮帐幕 |
| **无窗** | 巫族殿宇不开窗，光线由门洞及殿顶缝隙透入 |

### 图腾柱

| 项目 | 描述 |
|------|------|
| **材质** | 以巨兽骨骼或巨石雕刻而成 |
| **高度** | 约三丈至九丈不等 |
| **纹样** | 刻有各种野兽图腾（熊/虎/蛇/鹰/牛等），为各祖巫的氏族标志 |
| **数量** | 领地内共有图腾柱三十六根，围绕主殿排列 |

### 巨兽骨骼门

| 项目 | 描述 |
|------|------|
| **材质** | 以远古巨兽（如比山还大的巨犀/猛犸）的骨骼搭建 |
| **形态** | 两根骨骼作门柱，一根横梁搭在上面，形成"骨门" |
| **高度** | 高约五丈，宽约四丈 |
| **装饰** | 骨骼上刻有巫族符文，悬挂兽牙/兽爪作为辟邪物 |

## 装饰细节

### 祖火

| 项目 | 描述 |
|------|------|
| **位置** | 领地中央，常年不灭 |
| **颜色** | 血红色火焰，高约三丈 |
| **燃料** | 以巫族特选的"火精石"为燃料，散发原始热量 |
| **意义** | 象征盘古遗志，巫族生命之源 |

### 血池

| 项目 | 描述 |
|------|------|
| **位置** | 祖巫殿前，约方圆十丈 |
| **颜色** | 暗红色液体，非血非水，散发血腥与煞气 |
| **效果** | 祖巫在此修炼，可凝聚煞气入体 |
| **氛围** | 血池上方常有血雾缭绕，腥气冲鼻 |

### 祭台

| 项目 | 描述 |
|------|------|
| **位置** | 领地最高处，山顶 |
| **材质** | 以白玉巨石堆砌，祭台呈圆形 |
| **大小** | 直径约三十丈，中央有石台 |
| **纹样** | 台上刻有盘古开天图，以祖巫之血永久染红 |

## 重要地点详述

### 祖巫殿

- 十二祖巫聚会之地，殿内以血池为核心
- 殿中无像，唯有祖先石板

### 后土居所

- 位于领地东侧，与其他祖巫稍远
- 规模较小，建筑较为柔和

### 祭台

- 祭祀盘古之地，每年巫族大祭在此举行
- 云霄曾在此目睹后土主持祭礼
