# 👤 人物外观 · 女娲

> **⚠️ 风格统一：** Anime动漫风格 · 中国古典仙侠风 · 禁止混搭

## 整体气质
> 至高母神、仁慈威严、创造之美
>
> *"她的容貌无法用凡间言语形容——那是造化之美，是万物之母的慈悲，是天道的温柔一面。"*

### 👁️ 眼神

```
瞳色：澄金色瞳仁，蕴含日月精华，如天地初开时的第一缕光
眼神：母性的温柔中带着不可置疑的天威，是创造者独有的双重气质
眼形：杏仁状，眼尾上翘入鬓，有万种风情却不轻佻
眼波：眼波流转时，有星辰陨落之象，万物初生之美，令人无法直视太久
泪光：常含一滴怜悯之泪，凝视万物时，这泪光便化作创造之力
神采：慈祥中带威严，是母亲凝视孩提时的温柔，也是天道运转时的庄肃
```

### 👁️ 眉

```
眉形：弯月眉，眉如远山含黛，淡淡的青黑色，如水墨画中的留白
眉毛：根根分明，顺势而落，有自然生成之美，不加修饰却恰到好处
眉峰：眉峰浅淡，不显凌厉，反而柔和了眼睛的威严
动态：凝眉时轻锁，睁眼时舒展，变化间有鬼神造化之机
```

### 👃 鼻

```
鼻形：悬胆鼻，鼻梁高挺而不突兀，与面部轮廓相配恰到好处
鼻翼：鼻翼不大，张合有度，平静时纹风不动
鼻尖：圆润饱满，有大富贵之相，鼻孔含而不露
```

### 👄 唇

```
唇色：口唇如映日，红中透紫，是造化之力的颜色
唇形：樱桃小口，唇瓣饱满，微微一笑如春风拂面
上唇：薄而略带唇珠，人中浅窄而清晰
下唇：比上唇略厚，唇瓣柔和，有生育万物的母性光辉
笑时：笑时如大地回春，万物初始，让人心生安宁
```

### 🧏 耳

```
耳形：耳如元宝，耳垂厚实而饱满，有容纳万物之福相
耳色：与肌肤同色，温润如玉，透明处可见细红血络
佩戴：常戴一对泪滴形滴珠耳坠，羊脂玉所制，随步履摇曳
```

### 🐍 蛇身

```
整体：人首蛇身，上半身绝美女子，下半身为蛇尾盘旋
鳞片：鳞片晶莹如玉，泛着七彩灵光，赤金翠绿排列如虹
尾巴：蛇尾常蜷于足下，盘成一团，灵光流转其间
行动：行动时蛇尾轻摆，如龙游浅水，威严中透着柔美
```
