# 👤 人物外观 · 鸿钧道祖

> **⚠️ 风格统一：** Anime动漫风格 · 中国古典仙侠风 · 禁止混搭

## 整体气质
> 至高无上、威严如狱、深不可测
>
> *"他的存在本身就是天道——凡人不可直视，不可揣测，不可探究。"*

---

## 神话原型参考

- **《封神演义》**：鸿钧道人，为天道化身的最高存在
- **民间传说**：三清之师，天道圣人
- **地位**：洪荒世界最高存在，万圣之尊

## 容貌特征 · 详述（面部为主）

### 👤 脸型

| 维度 | 详细描述 |
|------|----------|
| 整体 | 苍老的面容，皱纹如刀刻斧凿，是无穷岁月留下的印记 |
| 肌肤 | 肌肤如古铜，透着紫金色泽，那是天道浸润的结果 |
| 骨骼 | 骨架极大，肩宽背阔，即便略显佝偻，也有压天之势 |
| 威压 | 面容常在金光中模糊，只现虚影，那是天道遮掩的威严 |

### 👁️ 眼神

```
瞳色：紫金色的瞳仁，深处疑有宇宙星河在运转
眼神：深不可测，看人时如看蝼蚁蚁穴，看穿一切生死贵贱
眼窝：眼窝深陷，常年罩在眉弓的阴影中，只有紫金光芒透出
眼形：三角眼，眼尾下垂，那是天地朽而道祖不朽的象征
直视：凡人不可直视，目光接融则神魂崩溃，如遭雷击
```

### 👁️ 眉

```
眉形：长寿眉，眉长过眼，垂落太阳穴两侧，如银丝卧龙
眉毛：根根如雪，晶莹放光，有天道符文若隐若现
眉峰：眉峰处有一道明显的横纹，那是无数次运算天道的痕迹
动态：眉动则天道变，眉静则万物宁
```

### 👄 须

```
胡须：银白长髯，飞垂过腹，如瀑布九天而落
胡形：胡须根根清晰，整齐中透着威严，不似老君的散乱
颜色：银白而非纯白，带着紫金光泽，是天道浸润的痕迹
修整：虽长却一丝不乱，常被老人以天道之力暗中梳理
```

### 👃 鼻

```
鼻形：伏羲大鼻，鼻梁如龙骨隆起，是天地开辟时就已铸就的骨相
鼻翼：鼻翼广大，不动声色时如临深渊，吸气则风云变色
鼻孔：鼻孔幽深，常有紫气从中泄出，那是吐纳天道的表现
```

### 🧏 耳

```
耳形：轮扇大耳，耳尖与头顶齐平，有容纳天地之相
耳垂：耳垂厚大，过肩而定，是\"福寿双全\"的极致之相
耳色：耳色与面皮同色，但内边缘常透紫金光晕
```

### 🔯 异象

- **头顶**：天花璎珞常悬于头顶，落地即散，放异香
- **金芒**：身周三尺有金紫色混元光环，明灭随心
- **混沌**：周身混沌气流环绕，时空在其身侧微微扭曲
- **中心**：其所在之处，灵气浓郁如水，他就是天地的中心
