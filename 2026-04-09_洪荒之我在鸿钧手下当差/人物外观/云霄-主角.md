## 容貌特征 · 详述（面部为主）

### 👤 脸型

| 维度 | 详细描述 |
|------|----------|
| 整体 | 瓜子脸型，轮廓如工笔白描，线条清秀而分明 |
| 饱瘦 | 颧骨微突，暂未发福，两颊带着常年风餐露宿的瘦削 |
| 太阳穴 | 微微凹陷，印堂不窄，反而显得精神内敛 |
| 下颌 | 线条利落，不似懦弱之辈，下巴略尖而翘 |
| 侧颜 | 侧颜如剑，棱角与圆润并存，有文人风骨而不失坚韧 |
| 风霜 | 面颊有一层淡淡的干晒红，那是行走洪荒留下的浅淡印记 |

### 👁️ 眼神

```
瞳色：深棕色，在光下略显金色，幽深如古井不起波澜
眼神：平静时如潭水不波，思考时如沉水有暗流涌动
眼形：杏仁状，眼尾微微上挑，眼尾长短适中，不过尖
眼窝：略深，眉弓的阴影投在眼下，显得深邃而难测
泪沟：下眼睑有一道淡淡的泪沟纹，片刻不睡即显青黑
神采：时而寒星般冷醒，时而暖阳般和煦，随心境而变
```

**眼的神韵：** 他的眼睛像是藏着一片看不见底的湖水——平静的表面下，你总觉得有什么东西在深处游动。

### 👁️ 眉

```
眉形：剑眉，眉尾斜飞入鬓，有男儿气却不显凶狠
眉色：黑色，与发色统一，偶尔有二三素毛横生
眉峰：眉心处有一道浅浅的悬针纹，常年忧虑凝结而成
眉毛：浓密程度中等，眉身比较顺，眉尾略有散乱
动态：思考时眉心会拧成一字，笑时眉尾会微微上飘
```

### 👃 鼻

```
鼻形：悬胆鼻，鼻梁挺直而不突兀，与面部轮廓相得益彰
鼻翼：鼻翼不大，张合有度，不动声色时唇形紧闭
鼻尖：鼻头略带圆润，不是尖耸的骨感，带几分柔和
鼻纹：鼻梁处有两道浅浅的横纹，常年皱眉所致
侧看：侧颜看鼻梁线条利落，是文人少有的立体轮廓
```

### 👄 唇

```
唇色：淡粉偏白，非常日烟火的血色，带着一丝营养不良的苍白
唇形：薄唇，唇线分明，不笑时常抿成一道细线
上唇：薄而略翘，唇珠不明显，人中浅窄
下唇：比上唇略厚，唇瓣松紧有度，收放自如
嘴角：微微上翘，笑时如淡墨起笔，带说不清的苦涩意味
唇纹：下唇中部有一道浅浅的横纹，常年忧虑而生的细纹
```

### 🦷 齿

```
整齐度：牙齿整齐白净，干净得不像常吃五谷之人
显露：说话时喜以唇齿音节为主，不开不扣
笑时：笑时露出三分上齿，礼貌而克制，学者气十足
```

### 🧏 耳

```
耳垂：耳垂单薄，不似有反骨之相，老人相书说是劳碌命
耳轮：轮廓分明，耳顶与眉尾齐平，耳前有细毛外透
佩戴：不戴耳环，只有几缕碎发垂落耳边，被风吹起时添几分疏狂
```

### 🖐️ 手部

- **手指**：修长白皙，指节分明，指腹有薄茧（执笔所致）
- **掌心**：纹路简单，命宫处有一道浅浅的事业线
- **习惯**：紧张时会无意识地用拇指轻搓腰间麻绳

### 🏃 体态

- **身高**：七尺有余，身形清瘦但不瘦弱
- **骨架**：肩宽适度，背脊挺直，有文人风骨
- **步伐**：稳健从容，每一步都在计算，从不浪费多余动作
- **姿态**：常有双手拢袖的习惯，那是收敛锋芒的姿态

## 服装设定 · 详述

### 👘 常服 · 素白道袍

```
材质：山野粗麻纺线织成，非丝绸之路的精细织物
颜色：月白色，日久泛黄，边缘处有风尘痕迹
质感：轻薄透气却不挺括，衣袂飘飘时有布衣的飘逸
气味：淡淡的草药与青茶香，混着一点竹叶的清苦
```

**款式细节：**
- 交领右衽，宽袖及肘，袖口有墨迹浸染
- 腰间麻绳以十字结系绑，松松垮垮，不加任何玉佩环佩
- 内衬红色细布裹身，领口露出一抹褪色红
- 下摆及踝，走路时微微飘动，不束缚行动
- **草鞋**：麻编草履，鞋底薄，走山路时轻便无声
