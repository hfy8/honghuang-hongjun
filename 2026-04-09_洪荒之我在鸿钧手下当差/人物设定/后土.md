# 👤 后土（祖巫）
> **⚠️ 风格统一：** Anime动漫风格 · 中国古典仙侠风 · 禁止混搭

## 基本信息

| 项目 | 内容 |
|------|------|
| 身份 | 十二祖巫之一、大地之母 |
| 外貌 | 威严中年女性形象，土黄色长裙，常有大地灵光环绕 |
| 本体 | 祖巫（盘古浊气所化） |
| 特技 | 祖巫化身、法天象地 |

## 人物设定

**核心性格：** 仁慈宽厚、顾全大局、外柔内刚

**背景：** 十二祖巫之一，被称为"大地之母"。在祖巫中唯一具有慈悲之心，也是云霄在巫族中最重要的朋友。

**关键矛盾：**
- 明知祖巫与妖族必有一战，却想保护族人
- 与云霄惺惺相惜，却站在对立阵营

**关键台词：**
- "这天地，不该只有强者才能生存。"
- "你我是友，但巫妖终有一战……我不愿看到那一天。"

## 外观描述（详细）

### 整体气质
至柔至刚、大地之母、慈悲与威严并存。整个人如同承载万物的大地，沉稳、厚重、包容万象。面容祥和却带着不可侵犯的威严，眼中有对众生的大慈悲，也有对命运的无奈。

### 容貌特征

| 部位 | 详细描述 |
|------|---------|
| **脸型** | 圆润饱满，颧骨微突，面容宽厚仁和，线条柔和而不失威严 |
| **眉** | 双眉如远山，眉尾下垂，眉峰柔和而非尖锐，眉色呈淡褐色，带岁月感 |
| **眼** | 眼眶大而深邃，瞳仁呈土黄色（大地之色），眸中常含慈悲，凝视时有众生皆在眼中的感觉；外眼角细长，眼睑柔和，眼周有细微皱纹 |
| **鼻** | 鼻梁不高但端宽，鼻翼厚实，鼻头圆润，有容人之量之相 |
| **唇** | 唇色淡红，唇线柔和，嘴角常微微上翘，带温和笑意；下唇略厚，唇纹细腻 |
| **耳** | 耳廓圆润，耳垂较长，佩戴一对土黄色玉环 |
| **肤** | 肤色呈健康的淡蜜色，触感如大地般温润，非白皙，带阳光感 |
| **须** | 下颌无胡须，为女性相，但面容不显女气，反而宽和端庄 |

### 发型发色

| 项目 | 描述 |
|------|------|
| **发色** | 深褐色，近似黑但带棕色润泽 |
| **长度** | 发长及腰 |
| **造型** | 将长发全数挽起，以一根粗银簪盘成低髻于脑后，额前有少量碎发 |
| **发饰** | 粗银簪一支，簪首雕有山岳纹；鬓边插一朵土金色小花 |

### 身形体态

| 项目 | 描述 |
|------|------|
| **身高** | 约八尺（约184cm），身量頎长 |
| **体型** | 丰腴健康，肩不宽但体态大方，腰身粗细适中，有母性之美 |
| **手** | 手指圆润，指节不突出，手掌宽大，有厚德载物之相 |
| **站姿** | 双手交叠于腹前，或双臂张开，姿态如大地承载万物，稳重如山 |

| 对敌时 | 眼神一凝，威严陡现，眉心竖纹显现 |
| 对云霄 | 目光柔和，带一丝惋惜与无奈 |

### 异象
- 身周常有土黄色大地灵光环绕
- 足下偶有绿草野花凭空生长
- 大地之力浓郁时，发间会有花瓣飘落
