## 人物设定

### 核心悬念（本书最大谜题）
- **为什么要收一个凡人为徒？**
- **"天定之人"是什么意思？**
- **云霄是棋子还是他真正看好的继承者？**

### 性格详解

| 维度 | 表现 |
|------|------|
| 深不可测 | 从不表露真实情绪，言行皆有深意 |
| 至高威严 | 圣人之下皆为蝼蚁，威压可令时空静止 |
| 算计无双 | 整个洪荒都在其棋盘之上 |
| 父师之心 | 对弟子有关心，但建立在"天道"框架之下 |
| 孤独 | 站在天道之上，无人可以理解 |

### 关键台词
- "你，上前来。"
- "你是天定之人。"
- "天数如此，不可改。"
- "这盘棋，我已经下了万亿年。"

## 外观描述（详细）

### 整体气质
至高无上、威严恐怖、深不可测、令人窒息的存在感。身周道韵流转，令时空扭曲，旁人无法直视其真容。整具身形如同一座不可撼动的高山，岁月在其身上留下的不是衰老，而是无尽的厚重。

### 容貌特征

| 部位 | 详细描述 |
|------|---------|
| **真容** | 不可描述，完整面容被金色道光完全遮掩，旁人只能看到模糊轮廓 |
| **常现相** | 面容苍老清癯，皱纹如刀刻，颧骨高耸，皮肤呈淡金色，仿若古铜雕塑 |
| **眉** | 双眉洁白如雪，长约三寸，眉尾垂落如银丝，眉心处有一道竖纹，威严毕现 |
| **眼** | 眼窝深陷，眶中双眸并非黑色，而是呈淡金或空白色，瞳孔如深渊，深不见底，目光扫过即令时空凝滞；眼周有金色光纹如年轮般扩散 |
| **鼻** | 鼻梁极高挺，鼻翼薄而紧，骨节分明，鼻下法令纹深刻如刀削 |
| **须** | 银白长髯垂落胸前，长约二尺，根根分明，如银丝瀑布，飘然若仙 |
| **发** | 纯白如银，发髻高束于顶，以一根墨玉簪横贯其中，发尾垂落背后，长及腰部 |
| **肤** | 呈古铜金色，触感如金石，非血肉之躯感，周身有淡金色光晕环绕 |

### 发型发色

| 项目 | 描述 |
|------|------|
| **发色** | 纯白如银雪，无一丝杂色 |
| **长度** | 发髻高束，发尾垂落及腰 |
| **造型** | 以墨玉簪挽成高髻，发丝根根清晰，无一丝凌乱 |
| **发饰** | 墨玉簪一支，通体漆黑，簪首雕有太极图案，长约五寸 |

### 身形体态

| 项目 | 描述 |
|------|------|
| **身高** | 约九尺（约207cm），远超常人 |
| **体型** | 清瘦但骨架宽大，肩宽背厚，骨架如铁，虽瘦却有无上威压感 |
| **手** | 手指修长枯瘦，骨节毕现，指尖有淡金色光芒，指甲呈半透明玉色 |
| **站姿** | 负手而立，或端坐于道台，从不倚靠，身形如松如岳 |

### 服装设定

#### 主服（紫色金纹道袍）

| 项目 | 描述 |
|------|------|
| **款式** | 宽袖广袂，交领右衽，衣身宽松，袖口垂落过膝，束腰宽幅 |
| **颜色** | 主色为深紫色，如黄昏暮霭，边缘以金线勾勒 |
| **纹样** | 整件道袍以金线绣满天道符文、云纹、星辰纹，肩部绣有北斗七星，衣摆绣有山河社稷图；后背正中有太极图直径约一尺 |
| **材质** | 天蚕丝，触手如玉，温润而沉，垂感十足，不染尘埃 |
| **层次** | 三层：内层雪白里衣，领口可见；中层紫色道袍；外层另有广袖外袍 |
| **腰带** | 金色丝绦腰带，宽约三寸，嵌有九宫八卦玉片，共九块 |
| **鞋** | 紫色云纹道履，鞋面绣有金线，鞋底厚约一寸，以檀香木制成 |

#### 头顶异象

| 项目 | 描述 |
|------|------|
| **天花** | 头顶常现天花坠落，花瓣晶莹剔透，呈淡金色，落地前即消散 |
| **祥云** | 身周有五色祥云缭绕，云中隐有龙凤虚影盘旋 |
| **金莲** | 足下常涌金莲，一朵一朵绽放又消散，步步生莲 |

| 配饰 | 位置 | 描述 |

| 九宫玉片 | 腰带 | 九块金色玉片，嵌于腰带，每块刻有一卦 |
| 天道玉蝶 | 头顶悬浮 | 约一尺直径的玉质蝴蝶，透明金色，悬浮于顶三尺处，缓缓旋转 |

### 法宝

| 法宝 | 形态 | 颜色 | 位置 | 描述 |
|------|------|------|------|------|
| 天道玉蝶 | 蝴蝶形态 | 透明金 | 头顶悬浮 | 约一尺直径，刻有完整天道，旋转时散发无穷法则之力 |
| 天地拂尘 | 拂尘 | 白银丝 | 常持左手 | 拂尘柄为墨玉，丝为银白，长约三尺 |

### 神态表情

| 状态 | 表情描述 |
|------|---------|
| 常态 | 面无表情，双眸如深渊，唇线平直，不见喜怒 |
| 说法时 | 偶有淡淡笑意，眉心竖纹微展，但转瞬即逝 |
| 威压时 | 眼睑低垂，威压外放，令在场者皆股栗欲坠 |
| 对弟子 | 目光微有柔和，唇角似有若无地上扬，但极为克制 |

### 特殊标记
- 眉心处有一道金色竖纹，约一寸长，非伤疤，乃天道凝聚之印记
- 周身永远笼罩淡金色光晕，如晨曦之日
- 足下生金莲步步绽放，印证其至高无上地位
