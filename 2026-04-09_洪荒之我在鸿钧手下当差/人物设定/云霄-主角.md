## 人物设定

### 背景来历
前世是一名普通上班族，熬夜加班后意外穿越到洪荒世界。带着未来20年的洪荒小说记忆（先知优势），成为道祖鸿钧的唯一凡人弟子。

**穿越身份：** 无系统、无老爷爷、无特殊血脉，只有"先知"这一最大金手指

### 性格详解

| 维度 | 表现 |
|------|------|
| 沉稳 | 无论遇到多大变故，脸上从不超过一丝波澜 |
| 隐忍 | 明明知道一切，却选择韬光养晦，不做出头鸟 |
| 足智多谋 | 每一步都在计算，利用先知优势布局 |
| 腹黑谨慎 | 明面上恭敬有礼，暗地里留三手准备 |
| 内心挣扎 | 知道太多却改变不了，知道命运却无法逃脱 |

### 关键人物关系

```
鸿钧（师尊）—— 敬畏、困惑、不确定是棋子还是继承者
后土（巫族）—— 惺惺相惜，终将站在对立面
帝俊（妖族）—— 互相试探，亦敌亦友
三清 —— 同门师兄弟，利益纠葛复杂
```

### 关键台词
- "天上从来没有白吃的午餐……"
- "我是棋子，还是下棋人？"
- "你们以为我在第一层，其实我在第五层。"
- "历史告诉我的，比你们想象的要多得多。"

### 成长线

1. **第一阶段（紫霄宫）**：小心翼翼求生存，收集信息
2. **第二阶段（巫妖量劫）**：开始小规模测试命运弹性
3. **第三阶段（圣人时代）**：逐渐介入诸圣博弈
4. **终极目标**：摆脱棋子命运，寻找自己的成圣之路

## 外观描述（详细）

### 整体气质
清逸出尘、隐忍内敛、书生气的凡人智者，眼神中永远藏着一抹看透世事的苦涩，嘴角偶尔浮现若有若无的苦笑。整个人如同一潭深水，表面平静，实则深不见底。

### 容貌特征

| 部位 | 详细描述 |
|------|---------|
| **脸型** | 清秀儒雅，轮廓分明如刀削，颧骨微突，面上略带风霜痕迹，介于青年与成年之间 |
| **眉** | 双眉修长如远山含黛，眉尾微微下垂，眉头紧锁时呈"川"字，舒展时如清风徐来 |
| **眼** | 眼眶略深，瞳仁漆黑如墨，偶有光芒流转如深潭；内眼角尖锐，外眼角微挑上扬，眼下有淡淡青黑（熬夜所致），眼神深邃平静却暗藏一丝苦涩 |
| **鼻** | 鼻梁挺拔但不过于高耸，鼻翼窄长，鼻头圆润略带肉感 |
| **唇** | 唇色淡红偏白，唇线分明，嘴角常微微内抿，形成若隐若现的苦笑弧度；下唇略厚于上唇 |
| **耳** | 耳廓分明，耳垂长度适中，不穿耳洞 |
| **肤** | 肤色白皙但非病态白，颧骨处有轻微日晒微红，近看有细微纹理，非无瑕瓷肌 |
| **须** | 下颌有淡淡青色胡渣，未刻意打理，约两至三天未刮，透出洒脱不羁感 |

### 发型发色

| 项目 | 描述 |
|------|------|
| **发色** | 纯正黑色，润泽光亮，在光线下略带青墨色调 |
| **长度** | 约过肩，发尾及肩胛骨处 |
| **造型** | 大部分时间松松束于脑后，以一根麻绳绑成低马尾，余发自然垂落肩侧；额前有两缕碎发垂落眉上 |
| **纹理** | 发丝柔顺但非丝缎质感，稍显毛躁，真实自然；鬓角处有两缕长发垂落耳侧 |
| **发饰** | 仅用一根灰褐色麻绳束发，无任何玉石金属装饰 |
