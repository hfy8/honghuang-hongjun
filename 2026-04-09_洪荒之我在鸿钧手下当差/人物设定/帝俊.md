# 👤 帝俊（妖族天帝）
> **⚠️ 风格统一：** Anime动漫风格 · 中国古典仙侠风 · 禁止混搭

## 基本信息

| 项目 | 内容 |
|------|------|
| 身份 | 妖族天帝、太阳之神、圣人大能 |
| 外貌 | 中年男子，面容威严而英俊，金色眼眸，散发帝王之气 |
| 本体 | 金乌（ 三足金乌） |
| 势力 | 妖族天庭 |

## 人物设定

**核心性格：** 帝王心术、城府极深、野心勃勃

**背景：** 帝俊是妖族的最高领袖，与弟弟太一共同建立天庭，欲统御洪荒。外表温文尔雅，实则心机深沉，擅长以退为进。

**关键矛盾：**
- 明知量劫将至，却无法放弃天帝之位
- 对云霄态度复杂——既忌惮其与鸿钧的关系，又想拉拢

**关键台词：**
- "天数如此，何不顺势而为？"
- "你我都不过是棋子罢了……但我要做那颗执棋的手。"

## 外观描述（详细）

### 整体气质
帝王风范、城府深沉、温润如玉而暗藏锋芒。整个人如同一潭静水，表面波澜不惊，实则深不见底。面容俊美中带着威严，举止间尽显帝王威仪，令人不敢直视。

### 容貌特征

| 部位 | 详细描述 |
|------|---------|
| **脸型** | 端正的国字脸，轮廓如雕如琢，颔宽阔，颧骨分明，线条硬朗又不失儒雅 |
| **眉** | 双眉如剑，漆黑如墨，斜飞入鬓，眉峰上挑但比太一平缓，眉心偶有浅纹，眉尾修长 |
| **眼** | 眶大而深邃，瞳仁呈赤金色，眸光深邃内敛，常有精光内敛不外放；外眼角微上挑，眼睑厚实，眼神如深渊，深不可测 |
| **鼻** | 鼻梁高挺如冠，鼻翼宽阔，鼻头圆润而不尖，有王者气度 |
| **唇** | 唇色淡红，唇线分明，嘴角常带温和笑意，唇瓣略薄，上唇有唇峰 |
| **耳** | 耳廓分明，耳垂厚实，佩戴一对金色龙纹耳环 |
| **肤** | 肤色呈健康的小麦色偏金，肌理细腻，面如冠玉，带日晒感 |
| **须** | 下颌蓄有三缕长须，约一寸半，整齐修剪，下巴正中一线垂下 |
