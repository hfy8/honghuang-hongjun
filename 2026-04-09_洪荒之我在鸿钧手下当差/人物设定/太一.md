# 👤 太一（妖族东皇）
> **⚠️ 风格统一：** Anime动漫风格 · 中国古典仙侠风 · 禁止混搭

## 基本信息

| 项目 | 内容 |
|------|------|
| 身份 | 妖族东皇、太阳之神、圣人大能 |
| 外貌 | 俊美青年男子，面如冠玉，金色眼眸，散发霸道之气 |
| 本体 | 金乌（ 三足金乌） |
| 势力 | 妖族天庭（与帝俊共有） |

## 人物设定

**核心性格：** 霸道直接、骄傲自负、重情重义

**背景：** 帝俊的弟弟，妖族天庭的武力担当。战力滔天，手中东皇钟镇压天际。与帝俊一文一武，共同执掌妖族。

**关键矛盾：**
- 性情刚烈，不屑权谋，但被迫卷入政治斗争
- 与云霄有过冲突，但英雄相惜

**关键台词：**
- "我东皇钟下，从无活口。"
- "兄长，你想太多了。有些事，一力降十会。"

## 外观描述（详细）

### 整体气质
霸道张狂、君临天下、太阳之威。整个人如同一轮炽烈燃烧的烈日，光芒万丈，不可直视。英俊的面容上永远带着张扬的自信，眉宇间尽是傲视苍生的霸气，令人见之不禁心生臣服。

### 容貌特征

| 部位 | 详细描述 |
|------|---------|
| **脸型** | 轮廓分明的俊美男子脸，颔宽阔，颧骨分明，下颌线如刀刻，整体呈端正的国字脸又不失俊秀 |
| **眉** | 双眉如剑，漆黑浓密，斜飞向上直入鬓角，眉峰如烈焰上挑，眉心无纹，偶有因怒而紧皱 |
| **眼** | 眶型锐利细长，瞳仁呈赤金色（太阳之色），眸中有烈焰在燃烧；外眼角极长且上挑，眼尾尖锐，眸光如电，扫之令万物焦枯；睁眼时常有金光外溢 |
| **鼻** | 鼻梁高挺如山岳，鼻翼紧而不厚，鼻头微尖，线条如神兵利刃 |
| **唇** | 唇色淡红，唇线分明，嘴角常带一抹狂傲的笑意；上唇薄于下唇，唇角有力上翘 |
| **耳** | 耳廓分明，耳垂大小适中，佩戴一对金芒耳钉 |
| **肤** | 肤色呈健康的小麦色偏金，触手如被日光炙烤过的温热，肌理细腻紧致 |
| **须** | 下颌光洁无须，年轻俊美，无任何胡须 |

### 发型发色

| 项目 | 描述 |
|------|------|
| **发色** |漆黑润泽，润如墨玉，在光线下有淡淡金棕色光晕 |
| **长度** | 发长及背脊 |
| **造型** | 将长发结成高马尾于头顶，以金冠束起，额前无刘海，露出王者额领；鬓角两缕长发垂落胸前，余发马尾垂于背后 |
| **发饰** | 金冠一顶（冠前沿有火焰纹金边）、金簪一支 |

### 身形体态

| 项目 | 描述 |
|------|------|
| **身高** | 约八尺二寸（约188cm） |
| **体型** | 精瘦有力，肩宽背阔，胸肌宽阔，腰身细窄，整体呈倒梯形，如同一柄出鞘的帝剑 |
| **手** | 手指修长有力，指节分明，右手虎口有薄茧（持东皇钟所致） |
| **站姿** | 双手叉腰，或单手撩起下摆，姿态张扬狂傲，重心常偏，常一脚踩石 |

| 状态 | 表情描述 |
|------|---------|
