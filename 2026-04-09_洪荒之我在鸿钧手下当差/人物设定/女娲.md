## 人物设定

### 核心性格：温柔慈爱但有不可触犯底线

| 维度 | 表现 |
|------|------|
| 表面 | 温柔婉约，母神般的慈悲 |
| 深层 | 护短之心极重，人族不可欺负 |
| 特点 | 创造与毁灭的双面性 |
| 底线 | 人族存亡是她唯一红线 |

### 背景
- 创造人族，成为人族守护神
- 巫妖两族的共同拉拢对象
- 圣人之位最早成就者之一
- 与妖族有血缘关系（兄长帝俊）

### 关键人物关系

```
女娲
  ├── 帝俊（兄长）—— 妖族天帝，血缘至亲
  ├── 人族 —— 创造之物，守护责任
  ├── 云霄 —— 欣赏其智慧，暗中关注
  └── 鸿钧 —— 同为圣人，相互尊重
```

### 关键台词
- "人族，是我所造，便由我所护。"
- "你很有意思，小云霄。"
- "兄长，你天庭之事，我不多管——但人族，不可动。"

## 外观描述（详细）

### 整体气质
至圣母神、温柔威严、创造之美。整个人如同一轮温润的明月，散发着柔和而不可忽视的光芒。上半身是绝美女子形态，下半身为蛇身，鳞片闪烁七彩，整体如同一件完美的艺术品。

### 容貌特征

| 部位 | 详细描述 |
|------|---------|
| **脸型** | 完美无瑕的鹅蛋脸，轮廓柔和流畅，颧骨不高，面部饱满如玉雕 |
| **眉** | 双眉如新月，弯而细长，眉色如墨，眉尾如远山含黛，眉心有淡金色花钿 |
| **眼** | 眶大而圆，瞳仁呈琥珀金色，眸光温润如春水；外眼角微上挑，眼睑如蝶翼，眸中常含慈爱与悲悯；凝视时如观众生 |
| **鼻** | 鼻梁纤细挺直，鼻翼小巧，鼻头圆润如珠，不施粉黛却完美无瑕 |
| **唇** | 唇色淡红如樱桃花瓣，唇线柔和，嘴角常带浅浅笑意，唇瓣丰润饱满 |
| **耳** | 耳廓玲珑，佩戴一对翠绿玉石耳坠 |
| **肤** | 肌肤胜雪，洁白如玉，触手如温润美玉，周身散发柔和光芒，非人间可有 |
| **须** | 无须，女性绝美形态 |

### 发型发色

| 项目 | 描述 |
|------|------|
| **发色** | 漆黑如墨，润泽如绸，在光线下有淡蓝色光晕 |
| **长度** | 发长及腰以下，垂落如瀑 |
| **造型** | 将长发挽成华髻于头顶，以红绳缠绕金步摇；额前有刘海如珠帘垂落；鬓边各有一缕长发卷曲垂落胸前 |
| **发饰** | 红绳一根、金步摇一支（垂珠以红宝石制成）、翠玉簪一支 |

### 身形体态（上半身为女子，下半身为蛇身）

| 项目 | 描述 |
|------|------|
| **身高** | 整体约一丈二尺（约276cm，含蛇尾） |
| **上身体型** | 女性身形，曲线优美，肩不宽但胸部饱满，腰身纤细，线条如瓶 |
| **蛇身** | 自腰部向下延伸为蛇身，粗约三尺，长约六尺；鳞片呈七彩之色，赤橙黄绿青蓝紫依次渐变，在光线下折射出绚丽光彩 |
| **手** | 手指纤细修长，指如削葱，佩戴一对翠玉镯，手持红绣球或宝莲灯 |
| **整体姿态** | 常以蛇身盘成圆盘状，上身直立，双手交叠于腹前或持物 |
