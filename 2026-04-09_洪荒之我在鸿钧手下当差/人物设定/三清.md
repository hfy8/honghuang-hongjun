## 人物设定

### 👴 老子
**核心性格：** 无为而治、仙风道骨、大智若愚

| 维度 | 表现 |
|------|------|
| 表面 | 淡然出世，不争不抢 |
| 深层 | 洞察一切，顺势而为 |
| 特点 | 不鸣则已，一鸣惊人 |

**背景：** 三清之首，鸿钧首徒。骑青牛西出函关，留下《道德经》传说。看似无为，实则看透天道运转。

**关键台词：**
- "道法自然。"
- "不争，故天下莫能与之争。"

---

### 👑 元始天尊
**核心性格：** 威严庄重、重视规矩、绝对权威

| 维度 | 表现 |
|------|------|
| 表面 | 一板一眼，等级森严 |
| 深层 | 秩序至上，不能容忍混乱 |
| 特点 | 正义凛然，但略显固执 |

**背景：** 三清之一，鸿钧次徒。乘九龙沉香辇，代表正统秩序。封神大战中支持阐教，与截教对立。

**关键台词：**
- "天数如此，当有尊卑。"
- "秩序，是天道之本。"

### ⚔️ 通天教主

| 维度 | 表现 |
|------|------|
| 表面 | 刚烈直接，不服就干 |
| 深层 | 重情重义，弟子如子 |

**背景：** 三清之一，鸿钧三徒。截教教主，门下弟子来自万族。封神大战中因弟子被害，怒而摆下万仙阵。

**关键台词：**

## 外观描述（详细）

### 👴 老子 · 外观详解

#### 整体气质
仙风道骨、超然物外、大智若愚。整个人如同一株遗世独立的老松，看似枯槁实则蕴含无穷生机。眼神浑浊却能洞察天道，容貌苍老却步履轻盈，不沾半点尘埃。

#### 容貌特征

| 部位 | 详细描述 |
|------|---------|
| **脸型** | 清瘦狭长，颧骨高耸，皱纹如网，眼窝深陷，面颊凹陷，典型的道家修行者苍老相 |
| **眉** | 双眉半白，眉长过目，眉尾垂落如银丝，眉头微微下垂，慈眉善目之相 |
| **眼** | 眼眶深陷，双眸小而深邃，瞳仁呈淡灰色（非全白），目光浑浊却仿佛能看透万物；眼睑皱纹层叠，笑时眯成一线 |
| **鼻** | 鼻梁高瘦，鼻翼极薄，鼻头略微下钩，有鹰喙之势 |
| **须** | 长髯垂胸，银白如雪，须长约二尺，根根清晰，飘然若仙，下巴处有三缕长须垂落 |
| **发** | 纯白如雪，披散于肩，发长及胸，发丝干枯但润泽，额前无碎发遮挡 |
| **肤** | 苍白如纸，干枯起皱，血管隐约可见，骨节突出 |

#### 发型发色

| 项目 | 描述 |
|------|------|
| **发色** | 纯白，无一丝杂色 |
| **长度** | 发长及胸，披散不束 |
| **造型** | 自然披散，额前无刘海，两鬓各有一缕长发垂落耳侧 |
| **发饰** | 无任何发饰，仅在鬓边插一朵白色小花 |

#### 身形体态

| 项目 | 描述 |
|------|------|
| **身高** | 约七尺八寸（约180cm），身量頎长 |
| **体型** | 极瘦，骨架分明，肩胛骨凸出，但腰背不驼，反而十分挺拔 |
| **手** | 枯瘦如柴，手指修长，骨节毕现，指甲略有青灰，持拂尘或牵牛缰绳 |
| **站姿** | 微微佝偻但不倒，负手而立，或倒骑青牛，逍遥自在 |
| **坐骑** | 青牛（青色大水牛），体型硕大，神态安详，牛角弯曲如弓 |

#### 服装设定

| 项目 | 描述 |
|------|------|
| **款式** | 最简素的道袍，交领右衽，宽袖不收口，衣身宽松 |
| **颜色** | 白色泛黄，如旧纸之色，非纯白 |
| **纹样** | 整件无绣花纹样，仅领口有一道灰色细边 |
| **材质** | 粗布，质地粗糙，触手有涩感 |
| **鞋** | 草鞋，麻绳编制，鞋面破旧 |
| **披风** | 灰色薄纱披风，夏日遮阳用，极简 |

#### 配饰清单

| 配饰 | 位置 | 描述 |
|------|------|------|
| 白色小花 | 鬓边 | 白色小花一朵，略有枯萎 |
| 扁拐 | 右手持 | 木制，黑色，长约三尺，顶端弯曲 |
| 拂尘 | 左手持 | 白色马尾为穗，柄为木制 |

### 👑 元始天尊 · 外观详解

#### 整体气质
威严庄重、不怒自威、至高无上。整个人如同一座黄金铸就的宝殿，华贵而不可侵犯。目光如电扫过即令天地失色，一举手一投足皆是天道正统的化身。

#### 容貌特征

| 部位 | 详细描述 |
|------|---------|
| **脸型** | 方正面庞，轮廓如刀刻，颧骨分明，面部饱满而不肥，满脸威严之色 |
| **眉** | 双眉漆黑如墨，粗浓斜飞入鬓，眉峰如剑，眉心有竖纹，威严毕现 |
| **眼** | 眶大而深，瞳仁漆黑如墨点，眸中常有冷电闪过，眼角上扬，眼睑厚实，不怒自威；下眼睑有细纹 |
| **鼻** | 鼻梁端正如山脊，鼻翼宽阔，鼻头圆润而大，鼻下人中深陷 |
| **须** | 短髯漆黑，约一寸，整齐修剪，呈"一"字形，下巴光洁无须 |
| **发** | 黑发高髻，发丝油亮，以金冠束起，鬓发整齐贴附，无一丝凌乱 |
| **冠** | 金色发冠，前额有金色额饰，嵌有紫玉，冠顶有一玉石卧狮 |

#### 发型发色

| 项目 | 描述 |
|------|------|
| **发色** | 漆黑润泽，无一丝白发 |
| **长度** | 仅及冠，余发皆束于冠内 |
| **造型** | 高髻束发，髻圆如珠，冠后有玉簪横贯 |
| **发饰** | 金冠一顶（嵌紫玉）、玉簪一支 |

#### 身形体态

| 项目 | 描述 |
|------|------|
| **身高** | 约八尺五寸（约195cm），十分高大 |
| **体型** | 健硕雄伟，肩宽背阔，胸肌饱满，腰细腿长，体态威严 |
| **手** | 手指粗壮，骨节分明，指腹有薄茧，右手常持如意或玉如意 |
| **站姿** | 正襟而立，脊背如铁，双脚与肩同宽，重心端正，不偏不倚 |
| **坐骑** | 九龙沉香辇（豪华车辇），由九条金龙牵引，车身以沉香木制成 |

#### 服装设定

| 项目 | 描述 |
|------|------|
| **款式** | 宽袖华服，交领右衽，袖口极宽大，衣身绣满纹样 |
| **颜色** | 金色为主，深青色为辅，领口为紫金双色 |
| **纹样** | 肩部绣日月星辰，胸前绣五爪金龙，衣摆绣海浪纹；全部以金线绣制 |
| **材质** | 上等丝绸，金线绣花，沉香木腰带 |
| **鞋** | 金色云纹履，鞋尖高高翘起，鞋面绣龙纹 |

#### 配饰清单

| 配饰 | 位置 | 描述 |
|------|------|------|
| 金冠 | 头顶 | 纯金，嵌紫玉，冠顶有玉石卧狮 |
| 玉簪 | 冠后 | 白玉制，长约四寸 |
| 玉如意 | 手中 | 白玉如意，柄刻祥云纹 |
| 龙纹玉佩 | 腰带 | 圆形白玉佩，雕五爪金龙 |
| 九龙辇 | 出行 | 沉香木车身，金色九龙拉车 |

### ⚔️ 通天教主 · 外观详解

#### 整体气质
叛逆张扬、锐利如剑、意气风发。整个人如同一柄出鞘的利剑，锋芒毕露，毫不掩饰。剑眉斜飞，眸光如电，嘴角常带一抹桀骜不驯的笑意，令人过目难忘。

#### 容貌特征

| 部位 | 详细描述 |
|------|---------|
| **脸型** | 轮廓分明如剑削，颧骨不高，下颌线锐利，整体呈倒三角形脸 |
| **眉** | 双眉如剑，漆黑浓密，斜飞向上直入鬓角，眉尾尖锐如刀锋，眉心竖纹因常皱眉而显 |
| **眼** | 眶型锐长，瞳仁漆黑如夜，眸中常有精光闪烁，眼尾极长且上挑，眼神锐利直接，常带挑衅之意；笑时眼如点漆，光芒慑人 |
| **鼻** | 鼻梁高挺笔直，鼻翼薄而紧，鼻头微尖，线条如刀刻 |
| **唇** | 唇色淡红，唇线分明，嘴角常上扬，带狂傲不羁之色；下唇略薄于上唇 |
| **须** | 下颌光洁无髯，偶有几根短须，不留胡须，年轻锐气 |
| **发** | 漆黑长发，额前剑分刘海，余发或束或散，鬓角两缕长发垂落胸前 |

#### 发型发色

| 项目 | 描述 |
|------|------|
| **发色** | 漆黑润泽，光可鉴人 |
| **长度** | 发长及腰 |
| **造型** | 额前刘海分成剑形左右分开；后发或束成高马尾，或披散于肩；鬓角两缕长发垂落胸前 |
| **发饰** | 无冠，以一根简单青玉簪束发，偶尔散发时亦不束 |

#### 身形体态

| 项目 | 描述 |
|------|------|
| **身高** | 约八尺（约184cm） |
| **体型** | 精瘦有力，肩不极宽但胸膛宽阔，腰身细窄，呈倒梯形，充满张力 |
| **手** | 手指修长有力，指节分明，右手常持青萍剑，左手指节处有握剑老茧 |
| **站姿** | 常单手叉腰，或抱手而立，重心常偏于一侧，姿态张扬不羁 |
| **坐姿** | 常半坐半靠，翘二郎腿，姿态散漫，与元始的端正截然相反 |

#### 服装设定

| 项目 | 描述 |
|------|------|
| **款式** | 玄色劲装，交领紧身，袖口收紧，束腰窄袖，便于战斗 |
| **颜色** | 主色为墨黑，边缘以青色丝线包边，领口为深青色 |
| **纹样** | 胸前背后各有阴阳鱼图；袖口与衣摆以青色回纹收边 |
| **材质** | 上等丝绸但偏向实用，垂感适中 |
| **披风** | 墨色披风，长及膝，无纹样，边缘以青线锁边 |
| **鞋** | 黑色武靴，靴筒高至脚踝，鞋底厚实，靴面有铜扣装饰 |

#### 配饰清单

| 配饰 | 位置 | 描述 |
|------|------|
| 青玉簪 | 发髻 | 青玉制，简单无纹，长约三寸 |
